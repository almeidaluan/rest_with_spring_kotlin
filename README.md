# DevFreela

## Features

- Buscar uma pessoa especifica do banco.
- Cadastrar uma pessoa.
- Listagem de pessoas.
- Atualizar uma pessoa.
- Remover uma pessoa.
- Upload de Arquivo.
- Multiplo Uploads de Arquivos.
- Download de Arquivo.

## Tech

Tecnologia e Patterns utilizados:

- [Kotlin - Spring Boot 3.0] - Backend!.
- [PostgreSQL] - Banco de dados.
